Foldio360 Animation with CSS

The following examples demonstrate how to achieve the effect of the animated pictures captured by the Foldio360 app, but using only CSS animation-* properties and @keyframes definitions.

Depending ou your needs, and what your tools allow, this MAY be sufficient to achieve independance from the spinzam.com service for showcasing your Foldio360 captures.

The Foldio360 image used in these examples were taken by my wife using the iPhone app and the 15" Foldio2 Studio. Here are the basic steps to get the image URL for reference or to be saved locally or to your web site/service...

Instructions:

  1. Use the Foldio360 App to capture and save your 360 image to spinzam.com
  2. Once available on spinzam.com, visit the provided URL for your captured 360 image, for example:
    https://spinzam.com/shot/?idx=1639
  3. Using your browswers "View Source" or "Inspect" tool, find the <div class="viewPort"> element, for example:
    <div class="viewPort" style="background-image: url("https://dc0sih4595mw5.cloudfront.net/contents/member/1283/12831474588621_film.jpg");">
  4. Copy and paste that URL into a new tab/window to see the full "film strip" version of your image:
    https://dc0sih4595mw5.cloudfront.net/contents/member/1283/12831474588621_film.jpg
  5. Save the image:
    • Locally: right-click -> "Save As"
    • Upload it to your web service or web site
    • Just reference it using the URL that spinzam.com uses (see step 3 above)
    • Optionally, convert the URL to a shorter one using a service like like tinyurl.com or goo.gl
  6. Now that you have the base image/strip of all frames of your 360 image, see the examples below on one way to use them without the need for JavaScript... using only CSS!
